// Munchkin Dictionary

Munchkin Dictionary is a modern Single Page Application (SPA) designed to provide a interactive experience for users looking to expand their vocabulary. By leveraging real-time data from a professional Dictionary API, it delivers definitions and usage examples through a high-impact, cinematic interface.
 
// Key Features in Dictionary API
    -Dynamic SPA Architecture: Search and retrieve data without ever refreshing the page, ensuring a fast and fluid user experience.

    -Real-Time Data Retrieval: Uses the Free Dictionary API to fetch up-to-date linguistic information including parts of speech and synonyms.

    -Audio Pronunciation: Integrated audio logic allows users to hear how words are spoken with a single click.

    -Interactive UI: Responsive search button and Enter key support.

    -Visual Design: A full-screen background with semi-transparent UI elements for a professional digital Library aesthetic.

    -Asynchronous Logic: Implements async/await and fetch() for non-blocking API calls.

    -DOM Manipulation: Programmatically creates and injects HTML elements into the page.

    -Event Handling: Uses sophisticated event listeners for both mouse and keyboard interactions.

    // MY PROJECT STRUCTURE

- index.html          = Semantic HTML structure and entry point.
- wordly-style.css    = Custom CSS for layout, themes, and typography.
- wordly-project.js   = The logic engine (API fetch, UI updates).
